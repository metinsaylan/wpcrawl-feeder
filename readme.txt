Official WPCrawl plugin to create a crawl feed for WPCrawl external wordpress crawler. Read more on <a href="https://wpcrawl.com/" target="_blank" rel="noopener noreferrer">WPCrawl</a>.

== Description ==

WPCrawl feeder plugin creates a backend for WPCrawl crawl requests to your site. It also helps you setup caching correctly to get the most out of your WordPress site.

Main Features:

* Site verification
* Set up secret Feed Key
* Manage crawler feed settings


== Installation ==

Use plugin search or upload plugin zip file to your server using "Plugins" "Add New" page.
Activate and visit Settings page to make necessary adjustments. Save plugin settings.
